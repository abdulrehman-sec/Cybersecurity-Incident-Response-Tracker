# Cybersecurity Incident Response Tracker

A **C language console application** for managing cybersecurity incidents. This tool allows security teams to **record, view, search, and delete incidents** efficiently.

## Features
- **Add Incident Records**: Store details like type, severity, reporter, date, status, and description.
- **View Incident Records**: Display all incidents in a structured format.
- **Search Incidents**: Quickly find incidents by their unique ID.
- **Delete Records**: Remove resolved or irrelevant incidents.
- **File-based Storage**: Persistent storage using binary files.

## Technologies Used
- **Programming Language:** C
- **File Handling:** Binary files
- **Platform:** Console application
